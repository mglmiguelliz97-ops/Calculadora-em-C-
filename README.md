# Calculadora-em-C-
🛠️ Minha Primeira Calculadora (C#) - Aprendizado na Raça
Este projeto é o resultado dos meus primeiros passos reais na programação. Não é um sistema complexo, é uma calculadora de console que fiz para parar de apenas ver teoria e começar a errar (e acertar) no código.
📝 O que eu passei para fazer isso:
Briga com a Sintaxe: Sofri bastante para entender onde abria e fechava cada chave {} e parêntese (). O código quebrou várias vezes por causa de um ; esquecido.
Lógica de Repetição: Tive dificuldade no começo para entender como manter o programa aberto usando o while sem travar o computador.
Condicionais: Aprendi na prática a diferença entre if, else if e como tratar erros bobos, como o usuário tentar dividir por zero.
⚙️ O que o código faz:
Soma, subtrai, multiplica e divide.
Tem um filtro para não deixar dividir por 0 (aprendi que isso trava o programa).
Só fecha quando você digita "sair", senão ele continua pedindo conta.
🚀 Por que postei isso?
Postei para documentar meu começo. Sei que o código é básico e que ainda erro detalhes de escrita, mas ele reflete o momento em que parei de ter medo do terminal e comecei a estruturar meu pensamento como um programador.
